# ClosetCoach

ClosetCoach is a cross-platform Expo/React Native MVP scaffold for an AI wardrobe planner.

## Included in this scaffold

- Mobile-first TypeScript Expo app
- MVP home experience with:
  - onboarding checklist
  - daily outfit suggestion
  - wardrobe summary
  - wear insights
- Starter component, feature, theme, and sample data organization

## Getting started

```bash
npm install
npm run start
```

Then press `i` for iOS simulator or `a` for Android emulator from the Expo CLI.

## Type checking

```bash
npm run typecheck
```
