# ClosetCoach

ClosetCoach is a cross-platform Expo/React Native MVP scaffold for an AI wardrobe planner.

## Included in this scaffold

- Mobile-first TypeScript Expo app
- Core loop v1 with:
  - local persistence via AsyncStorage
  - weather-aware outfit ranking (Open-Meteo)
  - optional read-only calendar inference with manual override
  - outfit swap controls and one-tap wear logging
  - wardrobe capture/edit/delete flow
  - wear insights from persisted logs
- Starter component, feature, theme, and sample data organization

## Getting started

```bash
npm install
npm run start
```

Then press `i` for iOS simulator or `a` for Android emulator from the Expo CLI.

## Type checking

```bash
npm run typecheck
```

## Unit tests

```bash
npm run test
```
